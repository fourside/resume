- 設計、実装、運用
- 主に管理画面(サーバサイド、フロントエンド)の作成
- アジャイル開発
    - イテレーション単位は2週間
    - JenkinsでCIを回す
        - JUnitで受け入れテストを作成

    - 使用した主なサービス: ec2, rds, cloudwatch, sns, ses
- バンドルしていなかったJavaScriptをビルドするようにした
